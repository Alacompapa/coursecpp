  int x[8];           x[3] = x[2];

       semantics:	x + 3 and x + 2 point to the 3rd and 2nd int beyond x.

pointer notation:	cout << *(argv + 2);
       semantics:	argv + 2 is used to select the location of element 2 of the array
			and the value of the element is reached using the deference operator(*).

       semantics:	x + 10 and x + 3 point to the 10th and 3rd int beyond x.

			3 and 10.
